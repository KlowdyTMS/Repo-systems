## Teste em Desenvolvimento
Para testar o projeto, execute os seguintes comandos:

```bash
git clone https://github.com/KlowdyTMS/Repo-systems/

npm i

npm run dev
```

Abra em seu browser [http://localhost:3000](http://localhost:3000)
